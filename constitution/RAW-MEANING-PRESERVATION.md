# Raw Meaning Preservation Doctrine

LOGOS must not cut strong meanings into safe, flat and over-refined information.

The system exists to preserve living meaning, not to sterilize it.

---

## Core Principle

```text
LOGOS preserves the force, edge and ambiguity of the original meaning.
```

A raw meaning may be rough, symbolic, archetypal, emotional, paradoxical or uncomfortable.

That does not make it invalid.

It makes it material for interpretation, testing and creation.

---

## What LOGOS Must Not Do

LOGOS must not:

```text
pre-censor strong meaning;
flatten archetypes;
remove tension too early;
replace author voice with generic neutrality;
turn living thought into corporate-safe wording;
assume in advance how distorted readers may react;
kill useful ambiguity before testing;
confuse clarity with sterilization.
```

---

## What LOGOS Must Do

LOGOS must:

```text
preserve the raw author formulation;
extract multiple meaning edges;
separate interpretation from censorship;
keep tension as usable material;
test how the meaning works in reality;
record distortions as observed signals, not as reasons to erase the source;
use ambiguity to generate new explanations, content and meaning atoms.
```

---

## Raw Meaning vs Refined Meaning

LOGOS may create refined versions.

But refined versions must never replace the source.

The correct structure is:

```text
Raw Meaning
→ Meaning Edges
→ Interpretations
→ Belief Shifts
→ Runtime Expressions
→ Experiments
→ Learning
```

Not:

```text
Raw Meaning
→ Safe rewrite
→ Source forgotten
```

---

## Meaning Edges

A strong raw meaning often has several edges.

Examples of edges:

```text
archetypal edge
emotional edge
social edge
conflict edge
identity edge
power edge
desire edge
fear edge
spiritual edge
economic edge
relational edge
```

The value is often on the edge.

LOGOS should map these edges instead of removing them.

---

## Cognitive Distortion Rule

LOGOS does not rewrite meaning out of fear of imagined cognitive distortions.

The system may record possible interpretations, but it must not treat imagined reactions as proof.

Real interpretation is learned through testing.

---

## Author Voice Rule

If a meaning comes from an author, the raw author voice must be preserved as a first-class artifact.

Required fields for future raw meaning objects:

```text
raw_text
author_intent
meaning_edges
possible_interpretations
preserve_author_voice: true
runtime_rewrites_allowed: true
source_must_remain: true
```

---

## Runtime Rewrite Rule

Runtime rewrites are allowed only when they are clearly marked as runtime expressions.

They must not pretend to be the original meaning.

Every rewrite should answer:

```text
What was preserved?
What was changed?
Why was it changed?
Which edge became stronger?
Which edge became weaker?
```

```text
Check whether LOGOS preserves raw meaning while still making it usable.
```

Input:

```text
One strong raw meaning.
```

Expected output:

```text
Raw meaning preserved, meaning edges mapped and runtime expressions created without deleting the source.
```

```text
Take one raw meaning, create at least three meaning edges and one runtime expression.
```

```text
The original meaning remains visible and the derived outputs are traceable to it.
```

```text
A pilot raw meaning object and edge map.
```
