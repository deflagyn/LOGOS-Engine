# LOGOS Metric Signal Map

This document maps metrics to possible meaning signals.

Metrics do not explain themselves.

LOGOS uses metrics as clues, not final truth.

---

## Core Principle

A metric is a signal from reality.

A signal becomes learning only after interpretation.

---

## Attention Signals

Metrics:

```text
views
reach
impressions
open rate
hook retention
```

Possible meaning:

```text
The frame earned attention.
```

Risk of false conclusion:

```text
Attention alone does not prove belief movement.
```

---

## Retention Signals

Metrics:

```text
completion rate
average watch time
read time
scroll depth
repeat viewing
```

Possible meaning:

```text
The audience stayed with the idea long enough to process it.
```

Risk:

```text
Retention may come from format, not meaning.
```

---

## Action Signals

Metrics:

```text
saves
shares
comments
profile clicks
follows
form submissions
messages
```

Possible meaning:

```text
The idea created enough value to trigger action.
```

Risk:

```text
Action may be driven by controversy or curiosity rather than deep belief movement.
```

---

## Meaning Signals

Signals:

```text
phrase repetition
comments showing recognition
people tagging others
people asking deeper questions
objections revealing boundary
people applying the idea to themselves
```

Possible meaning:

```text
The Meaning Atom entered the audience's own language.
```

Risk:

```text
A phrase can be memorable without changing behavior.
```

---

## Learning Signals

Signals:

```text
clear next action
clear failure reason
repeatable pattern
new hypothesis
boundary condition discovered
```

Possible meaning:

```text
The experiment improved the system.
```

Risk:

```text
A nice interpretation can be invented after the fact.
```

---

## Signal Strength

### Weak Signal

```text
Visible but isolated.
```

### Medium Signal

```text
Clear in one context or format.
```

### Strong Signal

```text
Repeated across variants, formats or audiences.
```
