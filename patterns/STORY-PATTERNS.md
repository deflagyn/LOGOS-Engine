# Story Patterns

Story Patterns are reusable narrative containers for delivering Meaning Atoms.

A Story Pattern should be simple enough to repeat and flexible enough to adapt.

---

## SP-001 — Mirror Reframe

### Structure

```text
Mirror / visible routine
→ hidden value
→ contradiction
→ reframe
→ emotional resolution
```

### Use Case

When the audience already accepts visible self-care but has not connected it to invisible health.

### Example Meaning Atom

> True self-care does not end at the surface.

---

## SP-002 — Mother to Daughter

### Structure

```text
What we teach daughters
→ what we forget to teach
→ new tradition
→ inherited care
```

### Use Case

When the goal is to transform a behavior into cultural inheritance.

### Example Meaning Atom

> Some forms of care should be inherited.

---

## SP-003 — Exhale After Action

### Structure

```text
Anxiety
→ uncertainty
→ reasonable action
→ calm
→ return to life
```

### Use Case

When the goal is to move the audience from worry to grounded action without promising certainty.

### Example Meaning Atom

> Peace begins when you know you did what was reasonable.

---

## SP-004 — Invisible Value

### Structure

```text
Visible value
→ invisible value
→ question
→ realization
```

### Use Case

When the goal is to elevate hidden health, trust, love or calm as more valuable than visible appearance.

### Example Meaning Atom

> Not everything valuable can be seen.

---

## SP-005 — Not a Lecture, a Thought

### Structure

```text
Personal observation
→ quiet question
→ human contradiction
→ reframe
→ one sentence to remember
```

### Use Case

For podcast-style videos where a doctor speaks not as a lecturer, but as a thoughtful guide.
